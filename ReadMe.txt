The file that you have been found in this repo is sourced from Kaggle.

This dataset has 2 columns 
1.) Height
2.) Weight

where Height is input variable
and Weight is Output variable.

Height is given in "Inches".
Weight is given in "Pounds".

As there is only 2 columns we are going with the Linear Regression model.

Problem statement :- 
Predicting the Weight of the given person based on their Height
